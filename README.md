# Estudo de Caso: Descomplicando ArgoCD

## Descrição

Este repositório contém materiais de estudo e exemplos práticos relacionados ao ArgoCD, uma ferramenta de entrega contínua para Kubernetes. O objetivo é entender o uso do ArgoCD, fornecendo recursos acessíveis para desenvolvedores e equipes de DevOps.
